UIs differ, but the required settings are usually the same.

## The 3 settings you need

1. **OpenAI API Key / API Key**: your su8 codes key
2. **OpenAI Base URL / Endpoint**: `https://su8.codes/codex/v1`
3. **Model**: a model id available to your account (copy from [Models](/en/tools/models/))

## How to verify quickly

- [API Tester](/en/tools/api-tester/)
- If errors happen: [Troubleshooting](/en/codex/troubleshooting/)
