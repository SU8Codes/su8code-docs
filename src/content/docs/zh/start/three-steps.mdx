**本文将引导您在最短时间内获取 SU8.Codes 的访问权限，并完成首次网络连通测试。**

## 1. 获取 API Key

1. 登录您的 [SU8.Codes 控制台](http://localhost:5434/console)。
2. 在“令牌”管理页面，点击**新建令牌 (Token)**。
3. 复制生成的令牌（形如 `sk-xxxxxx`）。

> **注意：** 令牌仅在生成时完整显示一次，请妥善保存在本地。若遗失请重新生成。

## 2. 测试 API 调用

请直接在下方粘贴您刚刚获取的 API Key 进行连通性测试。如果能正常返回 JSON 结果，说明通道已经为您完全打通。

<ApiTester client:visible />

## 3. 接入您的开发工具

拿到 Key 并测试成功后，您只需要在第三方工具（如 OpenClaw, Roo Code 或 Codex 客户端）的配置页面中填入以下两项：

1. **Base URL**: `https://www.su8.codes/codex/v1`
2. **API Key**: 填入刚保存的令牌

请前往 [接入总览](/zh/integrations/overview/) 选择您使用的工具，查看对应的保姆级图文教程。
