title: opencode 接入

opencode 支持通过“OpenAI Compatible Provider”接入任意 OpenAI 兼容网关（包括 su8 codes）。

## 在 opencode 里怎么填（只讲字段）

在项目目录创建/更新 `opencode.json`，重点填写：

- `options.baseURL`：`https://su8.codes/codex/v1`（从 `https://su8.codes/codex/v1/responses` 删掉 `/responses`）
- `options.apiKey`：`<你的 su8 codes API Key>`（或使用 `{env:SU8_API_KEY}` 从环境变量读取）
- `models`：把你要用的模型名写进去（可从 [模型列表](/zh/tools/models/) 复制）

示例（只保留关键字段，便于你对照填值）：

```json
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "su8": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "su8 codes",
      "options": {
        "baseURL": "https://su8.codes/codex/v1",
        "apiKey": "{env:SU8_API_KEY}"
      },
      "models": {
        "<模型名>": {
          "name": "<模型名>"
        }
      }
    }
  }
}
```
