Kilo Code supports “OpenAI Compatible”. To connect SU8 Codes, just fill the matching fields.

## What to fill in Kilo Code (fields only)

Choose “Use your own API key”, then fill:

- **API Provider**: OpenAI Compatible
- **Base URL**: `https://www.su8.codes/codex/v1`
- **API Key**: `<your SU8 Codes API key>`
- **Model**: `<model id>` (copy from [Models](/en/tools/models/))
