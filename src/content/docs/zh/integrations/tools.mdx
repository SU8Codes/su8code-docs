这页只讲一件事：**当工具让你填一堆字段时，你到底该填什么**。

## 一句话规则

- 工具里只要有 **Base URL / Endpoint / API Base** 这种字段：填 `https://www.su8.codes/codex/v1`
- 工具里只要有 **API Key / Token**：填你的 SU8 Codes key
- 工具里只要有 **Model / Model ID**：填你账号可用的模型名（去 [模型列表](/zh/tools/models/) 复制）

> 小心：有些工具的示例不带 `/v1`（比如示例是 `https://api.openai.com`）。  
> 你照样填完整的 `https://www.su8.codes/codex/v1`，不要自己脑补删路径。

## 常见字段对照（按人话）

- **Base URL**：就是“请求发到哪里”
- **API Key**：就是“你的通行证”
- **Model / Model ID**：就是“你要用哪个模型”
