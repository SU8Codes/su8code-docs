title: Kilo Code 接入
---

Kilo Code 支持 OpenAI Compatible。接入 su8 codes 时，你只需要把对应字段填好即可。

## 安装步骤

打开 VSCode，在扩展市场搜索 **Kilo Code** 并安装。

![Kilo Code 安装示意图](/integrations/kilo-code-install.png)

## 配置工具（字段怎么填）

选择 Use your own API key，然后填写：

- **API Provider**：OpenAI Compatible
- **Base URL**：`https://su8.codes/codex/v1`
- **API Key**：`<你的 su8 codes API Key>`
- **Model**：`<模型名>`（可从 [模型列表](/zh/tools/models/) 复制）
