title: Roo Code integration

Roo Code supports “OpenAI Compatible”. To connect su8 codes, just fill the matching fields.

## What to fill in Roo Code (fields only)

In Roo Code → Settings → Providers:

- **API Provider**: OpenAI Compatible
- **Base URL**: `https://su8.codes/codex/v1`
- **API Key**: `<your su8 codes API key>`
- **Model**: `<model id>` (copy from [Models](/en/tools/models/))
