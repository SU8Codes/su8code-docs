su8 codes implements an OpenAI-compatible API. If a tool supports **OpenAI Compatible** (custom Base URL + API Key), you can usually connect it quickly.

## Supported tools

| Protocol | Base URL | Tools |
| --- | --- | --- |
| OpenAI compatible | `https://su8.codes/codex/v1` | Cline, Roo Code, Kilo Code, OpenClaw, opencode, etc. |

> Note: If a tool’s placeholder does **not** include `/v1` (e.g. it shows `https://api.openai.com`), use `https://su8.codes/codex` (remove `/v1/responses` from `https://su8.codes/codex/v1/responses`).

## Notes

Before configuring a tool:

1. Create an **API Key** in your su8 codes dashboard.
2. Fill **Model / Model ID** with a model available to your account (copy from [Models](/en/tools/models/)).

Below are the exact fields to fill for each tool.

## Cline

### Install

In VSCode, search **Cline** in Extensions and install it.

### Configure (fields only)

In Cline → Settings → API Configuration:

- **API Provider**: OpenAI Compatible
- **Base URL**: `https://su8.codes/codex/v1` (remove `/responses` from `https://su8.codes/codex/v1/responses`)
- **API Key**: `<your su8 codes API key>`
- **Model ID**: `<model id>`

## OpenClaw

### Configure (fields only)

OpenClaw is primarily configured via `~/.openclaw/openclaw.json`. For a custom provider, fill:

- `baseUrl`: `https://su8.codes/codex/v1`
- `apiKey`: `<your su8 codes API key>`
- `api`: `openai-responses`
- `models[].id`: `<model id>`

Example (key fields only):

```jsonc
{
  "models": {
    "providers": {
      "su8": {
        "baseUrl": "https://su8.codes/codex/v1",
        "apiKey": "<SU8_API_KEY>",
        "api": "openai-responses",
        "models": [{ "id": "<Model_Name>", "name": "<Model_Name>" }]
      }
    }
  }
}
```

## opencode

### Configure (fields only)

Edit:

- `~/.config/opencode/opencode.json`

Fill:

- `options.baseURL`: `https://su8.codes/codex/v1`
- `options.apiKey`: `<your su8 codes API key>`
- `models`: add your model ids

Example (key fields only):

```jsonc
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "su8": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "su8 codes",
      "options": {
        "baseURL": "https://su8.codes/codex/v1",
        "apiKey": "<SU8_API_KEY>"
      },
      "models": {
        "<Model_Name>": { "name": "<Model_Name>" }
      }
    }
  }
}
```
