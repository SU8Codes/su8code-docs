title: Cline 接入
---

Cline 支持 OpenAI Compatible。接入 su8 codes 时，你只需要把对应字段填好即可。

## 安装步骤

打开 VSCode，在扩展市场搜索 **Cline** 并安装。

## 配置工具（字段怎么填）

在 Cline → Settings → API Configuration 中填写：

- **API Provider**：OpenAI Compatible
- **Base URL**：`https://su8.codes/codex/v1`（从 `https://su8.codes/codex/v1/responses` 删掉 `/responses`）
- **API Key**：`<你的 su8 codes API Key>`
- **Model ID**：`<模型名>`（可从 [模型列表](/zh/tools/models/) 复制）
