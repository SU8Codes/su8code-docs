title: 其他工具
---

su8 codes 已兼容 OpenAI 接口协议。只要第三方工具支持 **OpenAI Compatible**（自定义 Base URL + API Key），通常就可以快速接入使用。

## 适用工具

| 接口协议 | Base URL | 适用工具 |
| --- | --- | --- |
| 兼容 OpenAI 接口协议 | `https://su8.codes/codex/v1` | Cline、Roo Code、Kilo Code、OpenClaw、opencode 等 |

> 注意：有些工具的字段示例可能不带 `/v1`（例如示例是 `https://api.openai.com`），此时请填 `https://su8.codes/codex`（把 `https://su8.codes/codex/v1/responses` 里的 `/v1/responses` 一起删掉）。

## 说明

对编程工具进行配置前，建议先完成以下操作：

1. 在 su8 codes 后台创建 **API Key**。
2. 在工具的 **Model / Model ID** 字段填写你账号可用的模型名（不确定就去 [模型列表](/zh/tools/models/) 复制）。

下面按工具分别说明“在哪填 / 怎么填”。

## 接入 Cline

### 安装步骤

打开 VSCode，在扩展市场搜索 **Cline** 并安装。

### 配置工具

在 Cline → Settings → API Configuration 中填写：

- **API Provider**：OpenAI Compatible
- **Base URL**：`https://su8.codes/codex/v1`（从 `https://su8.codes/codex/v1/responses` 删掉 `/responses`）
- **API Key**：`<你的 su8 codes API Key>`
- **Model ID**：`<模型名>`

## 接入 Roo Code

### 安装步骤

打开 VSCode，在扩展市场搜索 **Roo Code** 并安装。

![Roo Code 安装示意图](/integrations/roo-code-install.png)

### 配置工具

安装完成后，配置以下信息：

- **API Provider**：OpenAI Compatible
- **Base URL**：`https://su8.codes/codex/v1`
- **API Key**：`<你的 su8 codes API Key>`
- **Model**：`<模型名>`

## 接入 Kilo Code

### 安装步骤

打开 VSCode，在扩展市场搜索 **Kilo Code** 并安装。

![Kilo Code 安装示意图](/integrations/kilo-code-install.png)

### 配置工具

选择 Use your own API key，然后填写：

- **API Provider**：OpenAI Compatible
- **Base URL**：`https://su8.codes/codex/v1`
- **API Key**：`<你的 su8 codes API Key>`
- **Model**：`<模型名>`

## 接入 OpenClaw

### 配置工具（字段怎么填）

OpenClaw 的配置文件通常在：

- `~/.openclaw/openclaw.json`

需要填写的核心字段：

- `baseUrl`：`https://su8.codes/codex/v1`
- `apiKey`：`<你的 su8 codes API Key>`
- `api`：`openai-responses`
- `models[].id`：`<模型名>`

示例（只保留关键字段，便于你对照填值）：

```jsonc
{
  "models": {
    "providers": {
      "su8": {
        "baseUrl": "https://su8.codes/codex/v1",
        "apiKey": "<SU8_API_KEY>",
        "api": "openai-responses",
        "models": [{ "id": "<Model_Name>", "name": "<Model_Name>" }]
      }
    }
  }
}
```

## 接入 opencode

### 配置工具（字段怎么填）

编辑 opencode 的配置文件：

- `~/.config/opencode/opencode.json`

重点填写：

- `options.baseURL`：`https://su8.codes/codex/v1`
- `options.apiKey`：`<你的 su8 codes API Key>`
- `models`：把你要用的模型名写进去

示例（只保留关键字段，便于你对照填值）：

```jsonc
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "su8": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "su8 codes",
      "options": {
        "baseURL": "https://su8.codes/codex/v1",
        "apiKey": "<SU8_API_KEY>"
      },
      "models": {
        "<Model_Name>": { "name": "<Model_Name>" }
      }
    }
  }
}
```
