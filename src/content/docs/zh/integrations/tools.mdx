---
title: 常用第三方工具怎么配
---

大部分第三方工具的设置长得不一样，但你只要抓住“同一套要填的东西”就行：

## 你要找的 3 个配置项

1. **OpenAI API Key**（或 API Key）：填你的 su8 codes key
2. **OpenAI Base URL**（或 Base URL / Endpoint）：
   - 填 `https://su8.codes/codex/v1`
3. **Model**：填一个你账号可用的模型名
   - 不确定就去 [模型列表](/zh/tools/models/) 复制

## 配完怎么确认是通的

先在文档站自检（最快）：
- [在线 API 测试](/zh/tools/api-tester/)

再回到第三方工具里点一次“测试/发送消息”，如果报错按这页排：
- [常见错误排查](/zh/codex/troubleshooting/)
