---
title: 安装 Codex（准备工作）
---

你要做到的效果很简单：**你的电脑上有一个能用的 Codex 客户端**（CLI 或编辑器插件都行）。

如果你已经能在终端里跑 `codex`，这页可以直接跳过，去下一页配置：
- [配置 su8 codes](/zh/codex/configure/)

## 你用哪种都可以

### 方案 A：Codex CLI
适合喜欢在终端里用的人。

你只要确认两件事：
- 终端能识别 `codex` 命令
- 你知道它的配置目录（通常在 `~/.codex/`）

### 方案 B：编辑器插件（比如 VSCode）
适合直接在编辑器里用的人。

你只要确认：
- 插件装好了
- 插件能让你配置“Base URL”和“API Key”（或者能读取环境变量）

> 不同版本的 Codex/插件安装方式可能不一样，这里不强行给你一条“唯一命令”。你装好后，关键是下一页把请求指向 `https://su8.codes/codex/v1`。
